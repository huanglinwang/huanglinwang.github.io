快捷键

发表于 2017-08-10 | 分类于 Other Base Technology

系统快捷键

Command + Shift + G 跳转到指定的文件夹
Command + K 连接服务器
Command + Shift + 4 截屏,可自定义
Command + Shift + 3 快速截全屏

Xcode 快捷键

Control + 6 列出当前文件中的所有方法,并且可以进行关键字过滤
Control + Command + Up/Down 在.h和.m文件之间切换
Command + Shifit + Y 隐藏或显示Console View
Command + 0 隐藏左边的导航区
Command + Option + 0 隐藏右边的导航区
Command + Control + Left/Right 定位到 上一次/下一次 编辑的文件
Command + Shift + F 在工程中查找
Command + . 结束本次调试
Command + T 新建一个 Tab 栏
Command + Shift + [ 在 Tab 栏之间切换
Command + Shift + 0零 快速打开苹果的官方文档
Command + Shift + O 快速定位到指定类的源码中

SDWebImage - 常用功能
Button隐藏点击高亮效果
