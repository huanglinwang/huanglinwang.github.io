导航栏 push 右上角黑边问题

发表于 2017-06-20 | 分类于 UI技巧

情景一 背景色引起的黑边问题

情景:
NavigationController 没有任何设置
VCOne 设置了背景颜色
VCTwo 没有设置背景颜色,不过 VCTwo 添加了一个 View 设置了颜色
此时就会出现黑边问题

解决办法

第一种
给 VCTwo 设置背景色即可
第二种
在 VCOne 中将 Navigation 的 NavigationBar的translucent属性设置为 false
